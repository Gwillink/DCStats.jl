DCStats.jl
=============

These IJulia notebooks provide the material used for the introduction to Julia presented at the [Statistical Programming DC Meetup](http://www.meetup.com/stats-prog-dc/events/138392652/). To follow along with them, you will need to install Julia, IJulia and several other packages. Instructions are found in the [Setup notebook](http://nbviewer.ipython.org/urls/raw.github.com/johnmyleswhite/DCStats.jl/master/Setup.ipynb).
